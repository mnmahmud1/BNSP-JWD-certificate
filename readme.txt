Deskripsi

Aplikasi Sederhana Menggunakan PHP dan Database MYSQL database, beberapa fiturnya :
- Siswa
    - Daftar
    - Login
    - Isi Formulir
    - Melihat Status Pendaftaran
    - Cetak Informasi Pendaftaran
    - Cetak Informasi Kelulusan
- Admin
    - Login
    - Lihat Informasi Pendaftar
    - Kelola Pendaftar
        - Ubah Status Pendaftaran
        - Detail Pendaftar
    - Kelola Akun Pendaftar
        - Ubah Password
        - Hapus Akun Pendaftar


Beberapa tools yang digunakan :
- PHP
- MYSQL

Software Yang digunakan :
- XAMPP for Windows V. 7.4.24
- Visual Studio Code V. 1.55.2
- Composer V. 2.1.9

CSS Framework yang digunakan :
- Bootstrap Code V. 5

Third Party yang digunakan :
- Datatables
- mPDF
